Пошук за місцем розташування
Пошук індексу здійснюється за територіальним розташуванням, шляхом вибору області до вибору конкретного району, населеного пункту. Пошук по вулиці здійснюється лише для обласних центрів, міст Київ, Севастополь та Сімферополь. Зверніть увагу, що для деяких населених пунктів в дужках може бути вказана приналежність до сільської чи міської ради

Пошук за назвою населеного пункту

X

Пошук по вулиці здійснюється лише для обласних центрів, міст Севастополь, Сімферополь та Київ!
Для здійснення такого пошуку необхідно обрати область, в полі "Оберіть район" обов’язково вказати "Обласний центр". При цьому поле "Введіть назву вулиці" стане доступним. Для того, щоби швидко знайти необхідну вулицю, варто вказувати перші 3-4 літери назви, не вводячи при цьому типу вулиці (проспект, провулок та інше).

Наприклад, за необхідності знайти індекс вулиці Поштової у місті Донецьк, параметри мають бути вказані так

Для усіх інших населених пунктів для пошуку індексів достатньо обрати лише назву населеного пункту.
